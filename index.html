    Advanced Analysis

    Audio echo detection, Whisper auto-captions, and audio-spike clip scoring. Requires a local proxy.

    OpenAI API key (for captions)

    Auto-captions use OpenAI's Whisper API. Cheap (~$0.003 per 30-second clip) and very accurate. Your key is stored only in this browser's localStorage — never sent anywhere except OpenAI.

    Security note: API keys in browser storage are vulnerable to XSS. This is fine for a personal tool you run locally. Don't paste your key on a shared machine. Create a dedicated key at platform.openai.com/api-keys and set a spending limit.

    Audio proxy configuration

    Twitch's CDN blocks direct audio fetching from web browsers (CORS). A small local proxy unlocks audio features. Run it in PowerShell while using the app.

    Setup instructions:
    1. Install Node.js (LTS) if you don't have it
    2. Download proxy.js (provided separately)
    3. Save it anywhere (e.g., Documents\stream-analyser)
    4. Open PowerShell in that folder and run node proxy.js
    5. Leave the PowerShell window open, paste the proxy URL below
